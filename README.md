# Texture-Pack-Template
Minecraft LCE template for texture-pack making.

If you use it:
- The icon of your pack will be the default one
- The name and the decription will be blank (idk why)

Recommended tools:
- .pck editor: https://github.com/LCERD/PCK-Studio
- .arc editor: https://github.com/PhoenixARC/MUArchiveEditor
  - .swf editor: https://github.com/jindrapetrik/jpexs-decompiler
- image editor: [https://libresprite.github.io/#!/](https://github.com/LibreSprite/LibreSprite)

Tutorial: [https://youtu.be/gxs1tqKz9Mo](https://youtu.be/gxs1tqKz9Mo)
